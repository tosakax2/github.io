リンク置き場

世の中に有能なサイトが多すぎて自分のブックマークが大変なことになってきたので、整理ついでにまとめておきます。

ナレッジコミュニティ

Zenn

https://zenn.dev/

クラスメソッド株式会社が運営するエンジニア向け情報共有コミュニティです。

Qiita

https://qiita.com/

Qiita株式会社が運営するエンジニア向け情報共有コミュニティです。

3DCG

忘却まとめ

https://bookyakuno.com/

Blenderのちょっと踏み込んだ情報がまとめられています。有用なアドオンの紹介もされているので定期的に訪れましょう。

Blender Market

https://blendermarket.com/

Blenderのアドオンはここから購入しましょう。頻繁にセールが行われています。

ambientCG

https://ambientcg.com/

CC0のテクスチャが大量に配布されています。

Poly Haven

https://polyhaven.com/

テクスチャだけでなくHDRI画像もダウンロードできます。

VRoid Hub

https://hub.vroid.com/

自作のVRMモデルを登録して動かすことが出来ます。

ドット絵

ドット道　復活版

http://temaemisokoubou.web.fc2.com/

かなり古いサイトですが、ここを見ればドット絵に必要な知識は殆ど得られると思います。

ドット絵作成講座

https://2d6.parasite.jp/dot_kouza/index.html

サクッと読めるのでドット絵をこれから描き始めるという人におすすめです。

コーディング全般

Github

https://github.co.jp/

コードの共有や共同開発ができます。

チームでソフトウェア開発する際にはもはや必須とも言えます。

paiza転職

https://paiza.jp/challenges/info

プログラミングスキルチェック問題を受けられます。

JavaDrive

https://www.javadrive.jp/

幅広に知識を身に着けたいならこのサイトが手っ取り早いでしょう。

Python

Python早見帳

https://chokkan.github.io/python/index.html

C++

ゼロから学ぶ C++

https://rinatz.github.io/cpp-book/

文字通り、ゼロからすべてを学ぶことが出来ます。

HTML/CSS

CodePen

https://codepen.io/

基本無料で使えるオンラインエディター。

HTML,CSS,JavaScriptを記述可能で、リアルタイムでプレビューを表示してくれます。

ブログサイトへの埋込みが簡単なので、コードの共有がしたくなったときにお世話になります。

web.dev

https://web.dev/?hl=ja/

HTML,CSS,JavaScriptの勉強ができます。

web開発に関する知識を身に着けるのに最適のサイトです。

配色

原色大辞典

https://www.colordic.org/

ブラウザで定義されているHTMLカラーネーム、日本の伝統色、世界の伝統色、Webセーフカラー、パステルカラー、ビビッドカラー、モノトーン、地下鉄のシンボルカラー、国鉄制定の塗装色の9つの色見本コンテンツがまとめられています。

色に迷ったらここからもらっていきましょう。

NIPPON COLOR

https://nipponcolors.com/

膨大な数の日本の伝統色が収録されています。

艦隊これくしょん

艦隊これ公式サイト

https://www.dmm.com/netgame/social/-/gadgets/=/app_id=854854

古いゲーム。

制空権シミュレータ v2

https://noro6.github.io/kc-web/

制空値を算出してくれるすごいやつです。

ぜかましねっと

https://zekamashi.net/

艦これ攻略系サイトの中で最も見やすくて信頼できると思います。

小説/SS

小説家になろう

https://syosetu.com/

言わずと知れた小説投稿サイト。まさに玉石混交といったところです。

ハーメルン

https://syosetu.org/

二次創作も一次創作もOKな小説投稿サイト。全体的に小説のクオリティが高いイメージがあります。
